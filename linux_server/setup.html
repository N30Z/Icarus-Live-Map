Pull Savegames docker cp

Copies all .json files from the Prospects directory inside the container into the local savegames/ folder. These files are used for cave, deposit, and geyser overlays on the map.

Memory Reader linux_reader.py

Test a single memory read of the Icarus process. The container must have Python 3 and the process must be running. Requires CAP_SYS_PTRACE or the container to run as root.
